# TypeScript in Action



## TypeScript Playground

> online REPL

https://www.typescriptlang.org/play/


## install

```sh
# macOS
$ npm i -g typescript

$ tsc -v
# Version 3.9.3

$ tsc -h

$ tsc --all

# Enable incremental compilation
$ tsc -i

```

## compile

```sh
# tsconfig.json
$ tsc --init

$ tsc helloworld.ts

```
